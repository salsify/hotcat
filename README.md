# Hotcat

Hotcat is a library for accessing and transforming ICEcat data into Salsify's required data ingest format.

## Installation

Add this line to your application's Gemfile:

    gem 'hotcat'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install hotcat
